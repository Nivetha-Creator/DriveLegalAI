# DriveLegal AI 🚦⚖️

## Overview

DriveLegal AI is an AI-powered road safety and traffic law assistance platform designed to help users understand traffic regulations, calculate challan penalties, and access legal guidance. The application aims to improve road safety awareness and simplify access to traffic-related information.

## Features

* Traffic Violation Analysis
* Challan/Fine Calculation
* Traffic Rules & Regulations Information
* AI-Powered Legal Assistance
* Interactive User Dashboard
* User-Friendly Web Interface

## Technologies Used

* Python
* Streamlit
* Pandas
* JSON
* Machine Learning
* Natural Language Processing (NLP)
* Git & GitHub

## Problem Statement

Many road users are unaware of traffic regulations and the penalties associated with violations. Accessing accurate legal information can be confusing and time-consuming. DriveLegal AI addresses this challenge by providing a centralized platform for traffic law awareness and guidance.

## Solution

The platform allows users to:

1. Learn traffic rules and regulations.
2. Identify violations and associated penalties.
3. Estimate challan amounts.
4. Receive AI-assisted legal guidance.
5. Improve awareness of safe driving practices.

## Project Structure

* `app.py` – Main Streamlit application
* `modules/` – Functional modules
* `data/` – Traffic rules and violation datasets
* `assets/` – Images and resources
* `requirements.txt` – Project dependencies
